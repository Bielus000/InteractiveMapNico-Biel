MOTIVACIÓ

La nostra motivació per crear aquesta web neix de la passió per combinar tecnologia i creativitat per construir experiències interactives úniques. Volem compartir el nostre treball i inspirar altres a explorar les possibilitats infinites del disseny digital i la narració visual immersiva.

OBJECTIU DE LA WEB

L'objectiu principal d'aquesta web és oferir una plataforma interactiva que permeti als usuaris explorar i descobrir contingut de manera innovadora. Busquem crear una experiència memorable que connecti amb l'audiència i demostri el potencial de les tecnologies web modernes per crear narratives immersives.

EINES UTILITZADES

Per desenvolupar aquest projecte hem utilitzat una combinació d'eines i tecnologies modernes, incloent HTML5, CSS3, JavaScript, i diverses llibreries de visualització. També hem emprat eines de disseny com Figma per a la planificació visual i GitHub per al control de versions i col·laboració en equip.
